# MURGRILL — Scrollytelling Website

Fiktive One-Page-Website für ein Burgerlokal aus Graz.
Scroll-linked Burger-Animation (WebP-Frames), Pop-Grill-Branding, Single-Line-Doodles, Reservierungstool.

## Lokal starten
```
python3 -m http.server 8000
```
→ http://localhost:8000

## Struktur
- `index.html` – die Website
- `brand-kit.html` – Brand Kit
- `sequence/` – Scroll-Frames (WebP)
- `images/` – Food- & Lokalfotos (WebP)
